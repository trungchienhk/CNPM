# Dự án Giao diện Website Bán Xe Mazda

Đây là dự án thiết kế giao diện cho một trang web giới thiệu và bán xe ô tô Mazda. Dự án được xây dựng hoàn toàn bằng HTML và CSS thuần, với cấu trúc nhiều trang liên kết với nhau.

##  Chức năng chính
* Trang Chủ (index.html): Hiển thị danh sách các dòng xe nổi bật (Mazda CX5, Mazda 3, Mazda BT50) cùng giá bán tham khảo.
* Trang Chi tiết sản phẩm: Mỗi dòng xe có một trang chi tiết riêng biệt, hiển thị hình ảnh hero, thư viện ảnh nội/ngoại thất và các phiên bản kèm giá bán cụ thể.
* Điều hướng mượt mà:Menu Navigation và các liên kết "Thông tin chi tiết" hoạt động trơn tru giữa các trang.
* Bố cục chuẩn: Sử dụng CSS Flexbox/Grid để căn chỉnh nội dung gọn gàng, giống với bản thiết kế mockup ban đầu.

## 🛠 Công nghệ sử dụng
* HTML5: Cấu trúc thành phần trang web.
* CSS3: Định dạng màu sắc, bố cục và font chữ.
* Trình soạn thảo: Visual Studio Code.
